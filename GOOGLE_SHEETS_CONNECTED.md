## 🔄 How It Works

1. **Student fills Google Form** → Data saved to Google Sheets
2. **Google Apps Script** → Converts sheet data to JSON
3. **Admin Portal** → Fetches JSON data automatically
4. **Dashboard & Tables** → Display real-time student data

## 🛠️ Troubleshooting

If data doesn't appear:

1. **Test the connection**: http://localhost:8000/admin/test-connection.html
2. **Check Google Apps Script**:
   - Make sure it's deployed as "Web app"
   - Access should be set to "Anyone"
   - Script should return JSON format
3. **Check browser console** for any errors (F12)
4. **Verify Google Sheets** has form responses
