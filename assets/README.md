# Assets Directory

This directory contains logos, icons, and other assets for the CompuNet Computer Center website.

## Current Files

- `logo.png` - Institute logo (used in navigation)
- `favicon.svg` - Website favicon
- `qr-placeholder.jpeg` - Google Pay QR code for payments

## Logo Files

The website uses `logo.svg` in the navigation. You can replace this with your actual institute logo:
- Recommended format: SVG or PNG
- Recommended size: 200x60px or similar aspect ratio
- Transparent background preferred

## QR Code

The `qr-placeholder.jpeg` is currently used on the payment page. To generate your actual Google Pay QR code:

1. Open Google Pay Business app
2. Go to "For Business" section
3. Select "QR Code"
4. Download your UPI QR code
5. Replace `qr-placeholder.jpeg` with your downloaded QR code
6. Keep the filename as `qr-placeholder.jpeg` or update the reference in `payment.html`

## Favicon

The `favicon.svg` is used as the website icon in browser tabs. You can replace it with your institute's icon.
